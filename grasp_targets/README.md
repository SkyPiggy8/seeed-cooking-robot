# 机械臂夹取对象清单

这个目录先放比赛现场会用到的可夹取物体。目标不是把所有厨房物体都列全，而是把目前适合 reBot Arm 练习抓取和摆盘的东西先整理出来，方便调相机、检测类别和菜谱。

第一批建议从这些物体开始：

- 番茄：圆形、颜色明显，适合测试识别和轻夹取。
- 鸡蛋碗：比单个鸡蛋更稳，适合做倒入动作。
- 汉堡：形状大、表面软，适合主食摆盘，但夹爪压力要小。
- 调料瓶：高度明显，适合测试竖直物体抓取。
- 碗、盘子、锅：先作为放置目标，不建议一开始直接抓。

现场摆放建议：

- 物体之间至少留 8 cm，避免检测框重叠。
- 汉堡放在盘子或垫纸上，不要直接贴在锅边。
- 软物体先用低速、低夹紧力测试。
- 不确定能不能抓稳的物体，先只跑 dry-run 看抓取点。
